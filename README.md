# React Todo

React temellerini ve **State Management** mantığını pekiştirmek için geliştirilmiş bir uygulamadır.

## Özellikler
- **CRUD:** Görev ekleme, silme ve düzenleme.
- **Teknoloji:** React (Hooks), CSS3, React Icons.
- **Amaç:** Vanilla JS ile kurulan mantığı modern React yapısına taşımak.
